Release History for Google Sitemap Providers

01.05.00 Release - 5th August 2011

  • 855 : Don't return multiple domains for a portal which uses multiple domains
  • 854 : Blog module 4.1 has changed method signature for GetBlogMonths(int,int)

01.04.00 Release

Bug Fixes

  • 714 : Blog provider not working with Blog module 4.0
  • 715 : Output blog module URls from permalink values instead of constructing urls

01.03.10 Release

Bug Fixes

  • 696 : Tagger provider always defaults yearly
  • 711 : Child portals not identified correctly, and have parent sitemaps generated instead

01.03.08 Release

Bug Fixes

  • 693 : websites with no locales enabled have no output values
  • 694 : ping function for verification of installation
  • 395 : index out of range error for News Articles provider

1.03.07 Release

Bug Fixes

  • 680 : If in sitemap index mode, empty sitemaps continue to be indexed
  • 678 : Forum sitemap returns null dates if no threads in a forum
  • 679 : Forum sitemap returns only 10 threads per forum, regardless of maxThredsPerForum setting

1.03.06 Release

Enhancements

  • 646 : DNN Forum 4.5 Support

Bug Fixes

  • 646 : Blog provider outputs duplicate urls when more than one blog post in a day

1.03.05 Release

Enhancements

  • 633 : Allow specification of pages to exclude with new 'excludePages' attribute
  • 614 : New Admin pages regex to filter out Admin tabs if needed
  • 635 : Use DNN 5.1 Page Priority value where available
  • 636 : Support News Articles short Url Format with new 'useShortUrls' attribute
  • 634 : Support News Articles secured format with new 'showSecured' attribute

Bug Fixes

  • 634 : News Articles output Secure articles in some versions
  • 613 : Mixed case portal alias can cause invalid urls to be output

1.03.02 Release

This release introduces the Forum provider as a standard provider for the set.

Enhancements

  • New Forum Module provider

1.03.00 Release

This release introduces the concepts of caching and sitemap index files, as well as giving greater control over the number of urls per sitemap file. It implements a new override for the SitePages() call to allow custom module providers to return a limited set of urls per sitemap.

Enhancements

  • Allow for sitemap index files (591)
  • Implement sitemap caching to speed output (590)

Bug Fixes

  • Increase sitemap limit to 50,000 urls (552)
  • News Articles add in language information into Urls (600)

1.02.14 Release

This release contains an update for the Ventrian Articles provider to version 07.03.

Bug Fixes / Enhancements

  • Fix duplicate entries when multi language portals with URl language parameter turned off (504)
  • Update Ventrian News Articles supported version to 07.03 (526)

1.02.12 Release

This release contains an update for the Ventrian Articles provider so that it supports multiple versions, plus a fix to the property agent to support the latest version.

Bug Fixes

  • Property Agent licensing code update to latest version
  • Update Property Agent supported version to 1.07.07
  • Output correct property agent 'agentType' parameters (508)
  • Change portal alias selection method (507)

1.02.08 Release

This release contains an update for the Ventrian Articles provider, plus a change to the Property Agent sitemap provider, so that it supports different settings of 'Search Status'.

Enhancements

  • Support latest version of Ventrian News Articles module by changing call to no-longer-supported method (500)
  • Change Property Agent sitemap to allow specification of the search status type (501)
  • Include instructions for IIS 7

1.02.07

Small Enhancement release - following up further from the changes in 1.02.06

Enhancements

  • Add in 'author' to the list of filters for News Article provider(473)
  • Add new overload for module providers to take specified locale and generate set of Urls with it (478)

Bug Fixes

  • Recognise when 'All' option is ticked in Category filters for News ARticles provider(473)
  • Output correct archive dates for Blog Provider (479)

1.02.06

Bug fix release. Thanks to Scott Schecter for pointing out bugs and providing fixes.

Bug Fixes

  • Case insensitive comparison on portal alias when running with Friendly Url Provider that doesn't set alias for .axd request types (ver 4.7 standard provider and up) (472)
  • Ventrian News Articles provider : use inbuilt title encoding function for article Urls. (444)

Enhancements

  • Ventrian News articles provider : automatically select best choice for retrieving articles based on category filters (473)
